# 信息整合

- **对普通人来说，很难再时间差上取胜，那么唯一能赢的地方就在于深度差** 
- **力求每一则信息都用自己话描述清楚，遇到问题时也想清楚最关键的问题是什么（用讲出来的方式）**
- **收集相关的信息，附上链接，并且用自己话概述里面最重要的信息**
- **寻找信息之间的关联，供需关系，以及市场倾向，最后是自己的洞察**
