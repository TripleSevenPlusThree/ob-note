# 分析框架

首先，分析客户群体，需求，价值，产品—— 
（同时也要分析其在研新品，新技术的价值，竞品等）
1. 企业提供差异化价值的产品或服务对客户的差异化价值在哪？目标客户群体的特质是怎样的？这种差异化价值下的产品的竞争本质是什么？品牌形象如何？（或者说产品形象）
2. 假设各种最坏的极端情况，例如替代品，技术变革，营销革新，要素成本上涨，经济情况，竞品等等，去分析：
    - 现有客户群体未来的变化情况（渗透率，边界，需求趋势等）？
    - 这种产品或服务被竞品替代的概率有多大？
3. 判断提供这种差异化价值的产品或服务的未来影响力会如何，包括未来需求趋势，及将其融合到各个领域，判断各个领域对其有没有需求，如果有该怎么创新产品去提供价值，即哪些领域在未来对其有需求。例如英伟达：人工智能新的实现方式对GPU并行计算能力的需求很大，使得英伟达的净利润在短短两年内从43.68亿增长到297.6亿，再到728.8亿，而2025年上半年已经赚了452亿）

其次分析商业模式，主要是产品组合、商业系统和战略——
1. 产品或业务组合实现了怎样的效果，打造了怎样的竞争优势？
2. 商业系统（业务系统、交易系统）和战略是如何实现这种差异化价值的？相比其他模式来说，是否具有竞争优势？竞争优势的可持续性如何（结合产品的本质价值、特性及竞争本质，商业模式，内部所做的事是否正确，以及外部发生的变化来判断）？

结合上述两点以及财务报表所评估的财务能力，
判断未来5到20年的盈利能力是良好且不断提升还是几乎停滞或每况愈下，即产生净现金流的能力如何。

接着分析核心理念，制度，管理，文化等等，以及管理层——
1. 企业的核心理念是什么？
2. 具有怎样的刺激进步的制度？是否与核心理念协调一致？
3. 组织架构是怎样的？
4. 管理方法是什么？如何实现协作与协同？
5. 激励制度如何？

最后根据看到的情况所对应的未来5到20年的内在价值是多少，即公司的市值或股价。
