# 电力行业

在简单看了国投电力，国电电力，长江电力的2023年年报后——
- 因为大型的电力公司都是国有控股，无周期性（除非电这种能源被彻底取代），重资产，所以
	- 所有电力公司的基本面都一致，负债高，经营现金流稳健，营运利润与增长也稳定，只要扩充规模，利润规模就会上去
	- 能不能有所增值关键就在于
		- **未来会不会扩张，装机规模会不会扩大**
		- **能源结构优化，主要以新能源为主**
		- **发展储能**
		- 抗恶劣灾害等风险能力
- 因此，**更应关注发展新能源的民营企业**，而关注国有控股的大型电力公司机会较少，除非因为市场情绪化波动而带来的机会
